# Sourabh Bhoyar — DevOps Engineer Portfolio

A modern, animated portfolio website showcasing DevOps engineering expertise in cloud infrastructure, Kubernetes, and Infrastructure as Code.

## 🚀 Features

- **Responsive Design** - Optimized for all devices with smooth animations
- **Modern Aesthetics** - Dark theme with animated gradient backgrounds and glassmorphism effects
- **Interactive Navigation** - Smooth scrolling and intuitive navigation menu
- **Code Showcase** - Featuring Kubernetes YAML examples
- **Contact Integration** - Easy way for visitors to get in touch

## 📋 Sections

- **Hero** - Eye-catching introduction with animated background orbs
- **About** - Overview of DevOps specializations (Infrastructure Automation, Kubernetes & Containers, CI/CD Pipelines, Cloud Platforms)
- **Experience** - Professional background and years of expertise
- **Projects** - Portfolio of completed work and achievements
- **Skills** - Technical expertise and tools mastery
- **Contact** - Get in touch section

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Advanced styling with CSS variables, gradients, and animations
- **Fonts** - Google Fonts (Inter, Space Mono)

## 📁 Project Structure

```
my-portfolio/
├── index.html          # Main portfolio page
├── style.css           # Styling and animations
└── README.md           # Project documentation
```

## 🎨 Design Highlights

- Custom CSS variables for consistent theming
- Animated background orbs with smooth transitions
- Glassmorphism design elements
- Gradient text effects
- Responsive grid layouts
- Smooth scroll behavior

## 🌐 Colors & Theme

- **Background**: Dark gradient (`#0a0e27` to `#1a1330`)
- **Accents**: Cyan (`#00d4ff`), Purple (`#6a7cff`), Pink (`#ff006e`)
- **Success**: Green (`#00d084`)

## 📱 Responsive Breakpoints

The portfolio is designed to work seamlessly across:
- Desktop (1180px+)
- Tablet (768px - 1024px)
- Mobile (< 768px)

## 🔧 How to Use

1. Clone or download this repository
2. Open `index.html` in a web browser
3. Customize the content with your information
4. Deploy to your preferred hosting platform

## 🚀 Deployment

This static portfolio can be deployed to:
- GitHub Pages
- Netlify
- Vercel
- Azure Static Web Apps
- Any web hosting service

## 📝 Customization

Edit `index.html` to:
- Update your name and role
- Add/modify experience and projects
- Change contact information
- Add social media links

Edit `style.css` to:
- Adjust color scheme via CSS variables
- Modify animations and transitions
- Update responsive breakpoints
